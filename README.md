# My first Workers AI

This is my first project using [Workers AI](https://developers.cloudflare.com/workers-ai/).

## Usage

```txt
git clone git@github.com:yusukebe/my-first-workers-ai.git
cd my-first-workers-ai
npm install 
npm run dev
npm run deploy
```
